## Build

### 0. Prerequisites

- Node.js
- [pnpm](https://pnpm.io/)

### 1. Install (first time only)

```sh
$ pnpm i
```

### 2. Start dev server

```sh
$ pnpm start
```

### 3. Build whole site

```sh
$ pnpm build
```
