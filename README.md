## Build

### 1. Install Gridsome CLI tool if you don't have

`yarn global add @gridsome/cli`

### 2. Start dev server

```sh
# first time only
$ yarn

$ yarn serve
```

### 3. Build whole site

```sh
$ yarn build
```
